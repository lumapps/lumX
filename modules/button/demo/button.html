Button

lx-button is a button directive that will become a link element if you specify a href or ng-href attribute.
Otherwise, it will become a button element.

Four different types of button are available: raised, flat, fab and icon. Disabled style will automaticly be applied when using ng-disabled or disabled attribute. Five different sizes are available and defined in dist/scss/core/settings/defaults.scss You can use color helpers to quickly create colored buttons.
You can set button color with main colors defined in dist/scss/core/settings/defaults.scss
Text/Icon color and ripple colors will be automaticly set according to the color. Or you can create a fully customizable button with background and text color helpers.
You need to use the css version of the button instead of the directive.
Moreover, you should specfify the ripple color by using the ripple directive.
Parameter Type Default Description
lx-size string m Define the button size. Options: xs, s, m, l, xl
lx-color string primary Define the button color. Options: colors defined in colors and sizes section in dist/scss/core/settings/defaults.scss
lx-type string raised Define the button type. Options: raised, flat, fab, icon
