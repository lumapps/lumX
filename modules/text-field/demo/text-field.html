Text field

lx-text-field is directive that create an input text element or a textarea element according to the transcluded element.

Parameter Type Default Description
lx-allow-clear boolean false Add a button to clear the input model.
lx-error boolean Set red error style.
lx-fixed-label boolean false Input label will act like a classic placeholder.
lx-icon string Add an icon next to the input. Input must have a fixed label in this case.
lx-label* string The input label displayed as a placeholder.
lx-theme string light Set this option to dark if inputs are displayed into a dark box. Options: light, dark
lx-valid boolean Set green valid style.
