Main colors

You can use color helpers to quickly create colored buttons.
You can set button background color with main colors defined in dist/scss/core/settings/_settings.defaults.scss.
Text/Icon color and ripple colors will be automaticly set according to the color.

Custom colors

Or you can create a fully customizable button with background and text color helpers. Moreover, you should specfify the ripple color.
